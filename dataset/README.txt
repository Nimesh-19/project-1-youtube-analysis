DATASET INFORMATION

The original dataset used in this project was too large to upload to GitHub.

Files used during analysis:

- US_Trending.csv
- IN_Trending.csv
- GB_Trending.csv
- CA_Trending.csv
- combined_youtube_data.csv (generated after data merging)

These files were used for:
- Data cleaning
- Exploratory Data Analysis
- Sentiment Analysis
- SQL analysis
- Tableau dashboard creation

To run this project locally, download the dataset from Kaggle:

https://www.kaggle.com/datasets/bsthere/youtube-trending-videos-stats-2026

After downloading:
Place the CSV files inside the dataset/ folder with the same names.
