# 📊 YouTube Trending Video Analysis

## 📌 Project Overview
This project analyzes YouTube trending video data across multiple countries (USA, India, UK, Canada) to uncover patterns in content performance, audience engagement, and sentiment.

---

## 🎯 Objectives
- Identify most popular video categories (genres)
- Analyze relationship between views and likes
- Perform sentiment analysis on video titles
- Compare engagement across regions

---

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Tableau
- NLTK (Sentiment Analysis)

---

## 🔄 Workflow

### 1. Data Collection
- Dataset downloaded from Kaggle
- Selected countries: USA, India, UK, Canada

### 2. Data Cleaning
- Removed duplicates
- Handled missing values
- Converted date formats

### 3. Analysis
- Category-wise performance
- Country-wise comparison
- Engagement metrics (views vs likes)

### 4. Sentiment Analysis
- Used NLTK VADER
- Classified titles into:
  - Positive
  - Neutral
  - Negative

### 5. Visualization
- Python charts (Matplotlib, Seaborn)
- Interactive dashboard using Tableau

---

## 📊 Key Insights
- Entertainment and Music categories dominate trending videos
- Strong positive correlation between views and likes
- Most video titles are neutral or positive
- Canada and USA show higher average engagement
- Monthly trends show variation in engagement across different time periods

---

## 📈 Dashboard Features
- Top categories by views

## 🧾 SQL Analysis
Key queries include:

## 📌 Conclusion
This analysis highlights how content category, audience region, and sentiment impact video performance. These insights can help creators and marketers optimize their content strategy.
